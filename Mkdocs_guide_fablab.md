Klon repoet

lav et Python Virtual Enviroment til mkdocs og plugins

https://realpython.com/what-is-pip/#using-pip-in-a-python-virtual-environment


```
python -m venv venv/
source venv/bin/activate

 pip3 --version


 pip --version

```

Bekræft at du er i dit Virtuel Enviroment og installer mkdocs 

``` bash
pip install mkdocs-material mkdocs-static-i18n
```

Nu burde du kunne bruge 
```
mkdocs build
mkdocs serve
```
og få et resultat eller i det mindste en fejl du kan søge på.
